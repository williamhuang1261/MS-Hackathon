# Certificate Templates

Place your certificate template PDFs in this folder:

- `purple-template.pdf` - Supporter Certificate ($10-24)
- `navy-template.pdf` - Community Builder Certificate ($25-99)
- `cream-template.pdf` - Safety Champion Certificate ($100-249)
- `gold-template.pdf` - Founder's Circle Certificate ($250-999)
- `platinum-template.pdf` - Changemaker Certificate ($1000+)

## Design Your Templates

Use **Canva** (free) to create beautiful certificates:

1. Go to canva.com
2. Search for "Certificate" templates
3. Choose an elegant design
4. Customize with Shield of Athena branding
5. Leave space for:
   - Donor name (centered, large)
   - Donation amount
   - Impact statement (4-5 lines)
   - Date and Certificate ID
6. Export as PDF
7. Place in this folder

See `/CERTIFICATE_SETUP.md` for detailed instructions.
